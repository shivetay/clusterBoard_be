# Cluster Board BE

Backend server for the Cluster Board application.

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- pnpm

### Installation

```bash
pnpm install
```

### Running the Server

```bash
pnpm dev - 'development'
```
